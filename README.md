Lumen – Knowledge Tracking & Learning Vault
🚀 Overview

Lumen is a web-based knowledge tracking application that helps you capture, organize, and retain what you learn from online content.

It allows users to save insights from YouTube videos and articles, organize them with tags, and track learning progress through simple analytics.

💡 Why I Built This

While learning from YouTube and articles, I realized most people consume content but don’t retain it.

So I built Lumen to:

Capture key learnings instantly
Organize knowledge in one place
Track consistency and learning habits
⚙️ How It Works
🔗 Input System

The app extracts preview data (like thumbnails for YouTube)

📝 Insight Capture

Notes (what you learned)
Tags (AI, Coding, Cloud, etc.)

💾 Data Storage
Uses browser localStorage
Saves all insights locally without backend

Includes:

Tag-based filtering

📈 Analytics
